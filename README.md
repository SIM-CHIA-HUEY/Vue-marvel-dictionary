## marvel-dictionary

This is a personal side-project that lists all the Marvel comic characters using Vue.js and external api of MARVEL.

## Project setup

__Preparations__

1. Git clone project
2. cd to project
3. ```npm install```

__Compiles and hot-reloads for development__

```
npm run serve
```

See [Configuration Reference](https://cli.vuejs.org/config/).
